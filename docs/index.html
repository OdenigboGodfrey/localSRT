Key Features

  • 100% Local: Your videos never leave your machine. No internet connection is required after setup.
  • Batch Automation: Easily select individual video files or process entire folders.
  • Configurable Performance: Tweak chunk sizes to balance your system's memory usage and processing speed.
  • Scalable AI Models: Choose between Tiny, Base, Medium, or Large transcription models based on your hardware capabilities.
⚠️ Notice: Because this application uses AI to generate subtitles, occasional inaccuracies or omissions may occur. Larger models generally provide better accuracy but take longer to process.

Run from Source

If you prefer to run LocalSRT directly using Python, clone the repository and install the required dependencies

Then execute your main application script to launch the elegant user interface.
